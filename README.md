# Derycash Frontend

Interface web do Derycash.

## Direcao inicial

- Experiencia principal: desktop.
- Responsividade: mantida desde o inicio, mas celular nao dita o layout principal.
- Stack: Next.js, TypeScript, Tailwind CSS, shadcn/ui, Radix UI, Lucide, Recharts, Anime.js, React Hook Form, Zod e Zustand.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
npm run typecheck
```

## Ambiente

Copie `.env.example` para `.env.local` e ajuste as variaveis.
